# Proassist Frontend (Next.js)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.local.example` to `.env.local`.
3. Start dev server:
   ```bash
   npm run dev
   ```

## Pages

- `/` Google signup entry
- `/app` JD input -> generate -> review -> send
- `/profile` user profile and resume upload
- `/history` generated/sent timeline
